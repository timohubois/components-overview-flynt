Get an overview of where components of the Flynt theme currently used.

== Description ==

Flynt Components Overview creates an admin menu page with a list table to get an overview of where components inside acf flexible content are currently used.

**Please note:** The active theme of the website must use the [Flynt Theme](https://flyntwp.com).

== Installation ==

= INSTALL WITHIN WORDPRESS =
(recommended)

1. Open **Plugins > Add new**
2. Search for **Flynt Components Overview**
3. Click **install and activate** the plugin


= INSTALL MANUALLY THROUGH FTP =

1. Download the plugin on the WordPress plugin page
2. Upload the ‘flynt-components-overview’ folder to the /wp-content/plugins/ directory
3. Activate the plugin through the ‘Plugins’ menu in WordPress


= AFTER ACTIVATION =

* By default: The homepage is immediately monitored and alerts are sent to the stored admin email.
